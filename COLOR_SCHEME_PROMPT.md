# CRM Color Scheme - Design System

## Primary Colors
- **Primary Dark**: `#60045d` - Main brand color for headers, buttons, navigation
- **Primary Light**: `#e30077` - Accent color for highlights, active states
- **Secondary**: `#5c186a` - Supporting color for secondary elements

## Usage Guidelines
- Use `#60045d` for main navigation, primary buttons, headers
- Use `#e30077` for active states, highlights, call-to-action elements
- Use `#5c186a` for secondary buttons, borders, supporting elements
- Maintain consistent color usage across all pages and components
- Apply gradients using these colors for modern UI elements

## CSS Variables
```css
:root {
  --primary-dark: #60045d;
  --primary-light: #e30077;
  --secondary: #5c186a;
}
```

## Implementation Notes
- Apply this color scheme to all dashboard pages
- Ensure consistent styling across team member dashboards
- Use these colors for charts, graphs, and data visualization
- Maintain accessibility with proper contrast ratios
