# Authentication & Credentials

This folder contains all login credentials and authentication information for the CRM system.

## Folder Structure

- `admin/` - Administrator login credentials
- `team/` - Team member login credentials  
- `api/` - API keys and tokens
- `database/` - Database connection credentials

## Security Notes

- Never commit actual credentials to version control
- Use environment variables for production
- Keep backup copies in secure location
- Rotate credentials regularly
